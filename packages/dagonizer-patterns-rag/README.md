# @noocodex/dagonizer-patterns-rag

LLM-driven node pattern bases for @noocodex/dagonizer (Decision, Compose, Scout).

## Install

```bash
npm install @noocodex/dagonizer @noocodex/dagonizer-patterns-rag
```

## Usage

See the [plugins guide](https://studnicky.github.io/Dagonizer/guide/plugins) and the [Archivist example](https://studnicky.github.io/Dagonizer/examples/the-archivist) for end-to-end composition.
