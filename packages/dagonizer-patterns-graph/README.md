# @noocodex/dagonizer-patterns-graph

Triple-store node pattern bases for @noocodex/dagonizer (Recall, Record, Digest).

## Install

```bash
npm install @noocodex/dagonizer @noocodex/dagonizer-patterns-graph
```

## Usage

See the [plugins guide](https://studnicky.github.io/Dagonizer/guide/plugins) and the [Archivist example](https://studnicky.github.io/Dagonizer/examples/the-archivist) for end-to-end composition.
