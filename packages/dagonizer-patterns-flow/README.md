# @noocodex/dagonizer-patterns-flow

Pure flow primitives for @noocodex/dagonizer (Select, Reduce, Gate, Extract, Respond).

## Install

```bash
npm install @noocodex/dagonizer @noocodex/dagonizer-patterns-flow
```

## Usage

See the [plugins guide](https://studnicky.github.io/Dagonizer/guide/plugins) and the [Archivist example](https://studnicky.github.io/Dagonizer/examples/the-archivist) for end-to-end composition.
